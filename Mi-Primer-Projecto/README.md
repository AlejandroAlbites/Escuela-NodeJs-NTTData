# Primer-Project

### Pasos para ejecutar

1. Clonar el repositorio
2. Instalar los modulos con `npm i`
3. Compilar los archivos typescript con `tsc`
4. Crear el archivo .env y agregar las Variables de entorno (
   PORT
   DB_URI  
    JWT_SECRET
   )
